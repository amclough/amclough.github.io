Maps, Measles, and Media

This project was completed during January of 2020 as part of the #CodeforPolicy2020 course under FiveThirtyEight's Dhrumil Mehta. A joint project with four students from both the Kennedy School of Government and Chan School of Public Health, we used the "have fun, go make cool data visualizations" prompt to create this PSA about the prevalence of measles in the U.S. and globally. The entire course was a whirlwhind 2 weeks from start to finish and ran scrum-style, serving as both a data visualization and programming bootcamp as well as a crash course in agile.

The project made use of twitter APIs, datawrapper and python-backed D3 visualizations, and github for version control.
